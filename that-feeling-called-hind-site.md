# That Feeling Called Hind Site

When we hate  

We want the object to feel  

To taste  

All the bitterness  

That confused state  

Hate ; To hate  

When we stop loving, to tempt fate  

All the bitterness  

The blood in the lake  

Crimson water glistening  

To taste  

All the hatred  

Where have we gone?  

That feeling called hind site  

To know it is a mistake  

When we hate.  

---

*Christopher MacGregor — 2013-12-20*
