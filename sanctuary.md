# Sanctuary

Having a place to hide  

Is a weakness  

Only limited time we have  

For sanding our edges smooth  

The weather is an effective tool  

So is love  

As is loss  

The one thing we need  

Is the one thing we avoid  

The weather is an effective tool  

So is love  

As is loss  

---

*Christopher MacGregor — 2016-05-21*
